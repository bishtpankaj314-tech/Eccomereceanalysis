# Ecommerce Business Analysis Using SQL

## Overview
SQL project analyzing ecommerce sales, marketing, customer segments, conversion rates, discounts, and delivery performance.

## Business Questions
1. Underperforming product categories
2. Marketing spend effectiveness
3. Conversion rate analysis
4. Valuable customer segments
5. Discount impact analysis
6. Delivery performance analysis
7. Growth KPI recommendations

## Tools
- SQL
- MsSQL
- GitHub

## Key Findings
- West region had high traffic but low conversion.
- Delivery delays impacted revenue.
- Marketing spend showed the strongest relationship with growth.
- Discounts did not consistently improve revenue.
